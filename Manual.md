# Snake Game - 사용 설명서

## 게임 소개

클래식한 뱀 게임(Snake Game)을 Python과 Pygame으로 구현한 프로젝트입니다. 
뱀을 조종하여 먹이를 먹고 점수를 획득하며, 벽이나 자기 몸에 부딪히지 않도록 조심하세요!

## 게임 특징

- **레트로 스타일 그래픽**: 클래식한 뱀 게임의 느낌을 살린 그리드 기반 디자인
- **난이도 시스템**: 점수가 높아질수록 뱀의 속도가 빨라집니다
- **콤보 시스템**: 3초 이내에 연속으로 먹이를 먹으면 보너스 점수와 반짝이는 효과
- **최고 점수 저장**: 게임을 종료해도 최고 점수가 저장됩니다
- **사운드 및 음악**: 게임 이벤트에 맞춰 효과음과 배경음악 재생
- **게임 통계**: 플레이 시간, 먹은 먹이 수 등 상세한 통계 제공

## 시스템 요구사항

- Python 3.7 이상
- Pygame 라이브러리
- NumPy 라이브러리 (사운드 생성용)

## 설치 방법

1. 필요한 패키지 설치:
```bash
conda run -n pygame pip install -r requirements.txt
```

2. 게임 실행:
```bash
conda run -n pygame python main.py
```

## 조작법

### 기본 조작
- **방향키 (↑ ↓ ← →)**: 뱀의 이동 방향 변경
- **SPACE**: 일시정지/재개
- **ESC**: 게임 종료

### 게임 오버 후
- **R**: 게임 재시작
- **ESC**: 게임 종료

### 사운드 조작
- **S**: 효과음 켜기/끄기
- **M**: 배경음악 켜기/끄기

## 게임 규칙

### 기본 규칙
1. 뱀은 항상 앞으로 이동합니다
2. 먹이를 먹으면 뱀의 길이가 1칸 늘어납니다
3. 먹이 하나당 10점을 획득합니다
4. 벽에 부딪히거나 자기 몸에 부딪히면 게임 오버입니다

### 콤보 시스템
- 10초 이내에 연속으로 먹이를 먹으면 **콤보 활성화**
- 콤보 활성화 시 +5점 보너스 점수 획득- 10초가 지나면 콤보가 자동으로 해제되며 시각 효과가 사라짐- 콤보 활성화 시 시각 효과:
  - 상단 패널 중앙에 "COMBO!" 텍스트가 주황색으로 반짝임
  - 뱀 전체가 밝게 빛나며 노란색 톤이 추가됨

### 난이도 상승
점수가 올라갈수록 게임 속도가 증가합니다:
- 0-100점: 10 FPS (느림)
- 100-300점: 12 FPS
- 300-500점: 14 FPS
- 500-800점: 16 FPS
- 800점 이상: 18 FPS (빠름)

## 화면 구성

### 상단 정보 패널
게임 화면 상단에는 다음 정보가 표시됩니다:
- **Score**: 현재 점수
- **High**: 최고 점수
- **Length**: 뱀의 길이
- **Speed**: 현재 게임 속도 (FPS)
- **S**: 효과음 상태 (ON/OFF)
- **M**: 배경음악 상태 (ON/OFF)
- **COMBO!**: 콤보 활성화 시 화면 중앙에 주황색으로 반짝이며 표시됨

### 게임 영역
- **녹색 사각형**: 뱀 (밝은 녹색 = 머리, 어두운 녹색 = 몸통)
- **빨간색 사각형**: 먹이 (크기가 변하는 펄스 효과)
- **회색 그리드**: 게임 영역 구분선

### 게임 오버 화면
게임 오버 시 다음 정보가 표시됩니다:
- 최종 점수
- 최고 점수 또는 신기록 표시
- 먹은 먹이 개수
- 플레이 시간

## 팁과 전략

1. **콤보 유지하기**: 10초 안에 먹이를 먹어 콤보를 유지하면 더 높은 점수를 획득할 수 있습니다
2. **공간 확보**: 뱀이 길어질수록 움직일 공간이 줄어들므로 공간을 효율적으로 사용하세요
3. **앞을 내다보기**: 빠른 속도에서는 미리 경로를 계획하는 것이 중요합니다
4. **벽과의 거리**: 벽에 너무 가까이 가지 않도록 주의하세요
5. **연습**: 처음에는 느린 속도로 게임에 익숙해지세요

## 문제 해결

### 게임이 실행되지 않을 때
- Python과 필요한 라이브러리가 제대로 설치되었는지 확인
- Pygame 버전이 호환되는지 확인 (2.0 이상 권장)

### 사운드가 들리지 않을 때
- S 키를 눌러 효과음이 켜져 있는지 확인
- M 키를 눌러 배경음악이 켜져 있는지 확인
- 시스템 볼륨 설정 확인

### 게임이 느리게 실행될 때
- 다른 프로그램을 종료하여 시스템 리소스 확보
- Python 버전과 Pygame 버전이 최신인지 확인
