# Snake Game - Manual (Version 3.0)

## 게임 소개

클래식한 뱀 게임(Snake Game)을 Python과 Pygame으로 구현한 프로젝트입니다.
Version 3.0에서는 **난이도 시스템**, **장애물**, **Portal 모드**, **Golden Apple** 등 새로운 기능이 추가되었습니다!

## 게임 특징

- **3가지 난이도**: Easy / Normal / Hard
- **장애물 시스템**: Hard 난이도에서 5개의 장애물 등장
- **Portal 모드**: 벽을 통과하여 반대편으로 이동
- **Golden Apple**: 50점을 주는 특수 먹이 (10% 확률)
- **콤보 시스템**: 10초 내 연속 먹이 섭취 시 보너스
- **설정 저장**: 난이도, 포탈 모드, 사운드 설정 자동 저장
- **최고 점수 기록**: 게임 종료 후에도 유지

## 시스템 요구사항

- Python 3.8 이상
- Pygame 2.0 이상
- NumPy (사운드 생성용)

## 설치 및 실행

```bash
# 패키지 설치
conda run -n pygame pip install -r requirements.txt

# 게임 실행
conda run -n pygame python main.py
```

## 조작법

### 기본 조작
- **방향키 (↑ ↓ ← →)**: 뱀의 이동 방향 변경
- **SPACE**: 일시정지/재개
- **ESC**: 게임 종료

### 게임 오버 후
- **R**: 게임 재시작
- **ESC**: 게임 종료

### 설정 조작
- **D**: 난이도 변경 (Easy → Normal → Hard → Easy)
- **P**: Portal 모드 토글 (ON/OFF)
- **S**: 효과음 켜기/끄기
- **M**: 배경음악 켜기/끄기

## 난이도 시스템

### Easy (쉬움) - 초록색 [E]
- **초기 속도**: 8 FPS (매우 느림)
- **속도 증가**: 먹이 10개마다 +1 FPS
- **최대 속도**: 15 FPS
- **장애물**: 없음
- **추천**: 처음 플레이하는 분

### Normal (보통) - 노란색 [N]
- **초기 속도**: 10 FPS (기본)
- **속도 증가**: 먹이 5개마다 +1 FPS
- **최대 속도**: 18 FPS
- **장애물**: 없음
- **추천**: 일반 플레이어

### Hard (어려움) - 빨간색 [H]
- **초기 속도**: 12 FPS (빠름)
- **속도 증가**: 먹이 3개마다 +1 FPS
- **최대 속도**: 22 FPS
- **장애물**: 5개 (회색 블록)
- **추천**: 숙련된 플레이어

## Portal 모드

Portal 모드를 켜면 **벽을 통과**할 수 있습니다!

- **왼쪽 벽** → 오른쪽 끝에서 나타남
- **오른쪽 벽** → 왼쪽 끝에서 나타남
- **위쪽 벽** → 아래쪽 끝에서 나타남
- **아래쪽 벽** → 위쪽 끝에서 나타남

**주의**: 자기 몸통 충돌은 여전히 게임 오버입니다!

상단 패널에 **"P"** 표시가 있으면 Portal 모드 활성화 상태입니다.

## Golden Apple (금 사과)

일반 먹이 대신 **노란색 Golden Apple**이 나타날 수 있습니다!

- **출현 확률**: 10% (일반 먹이 10개당 1개 꼴)
- **점수**: +50점 (일반 먹이의 5배!)
- **지속 시간**: 10초 후 일반 먹이로 변경
- **외형**: 금색으로 반짝임

Golden Apple을 놓치지 마세요!

## 장애물 시스템

Hard 난이도에서는 **회색 장애물** 5개가 랜덤 위치에 등장합니다.

- **충돌 시**: 즉시 게임 오버
- **위치**: 게임 시작 시 랜덤 생성 (뱀/먹이와 겹치지 않음)
- **재시작 시**: 새로운 위치에 재생성

전략적으로 움직여 장애물을 피하세요!

## 콤보 시스템

**10초 내**에 연속으로 먹이를 먹으면 콤보 활성화!

- **보너스 점수**: +5점 추가 (일반 먹이 기준)
- **시각 효과**:
  - 상단 패널 중앙에 "COMBO!" 텍스트 (주황색 반짝임)
  - 뱀 전체가 밝게 빛나며 노란색 톤 추가
- **자동 해제**: 10초가 지나면 콤보 종료

Golden Apple은 콤보 점수가 적용되지 않습니다.

## 화면 구성

### 상단 정보 패널
- **Score**: 현재 점수
- **High**: 최고 점수
- **Len**: 뱀의 길이
- **Spd**: 현재 속도 (FPS)
- **[E/N/H]**: 난이도 (Easy/Normal/Hard)
- **P**: Portal 모드 활성화 여부
- **S**: 효과음 활성화 여부
- **M**: 배경음악 활성화 여부
- **COMBO!**: 콤보 활성화 시 중앙에 표시

### 게임 영역
- **밝은 녹색**: 뱀 머리
- **어두운 녹색**: 뱀 몸통
- **빨간색**: 일반 먹이 (펄스 효과)
- **금색**: Golden Apple (반짝임)
- **회색**: 장애물 (Hard 난이도)
- **검은색**: 배경
- **회색 그리드**: 게임 영역 구분선

### 게임 오버 화면
- 최종 점수
- 최고 점수 또는 신기록 표시
- 먹은 먹이 개수
- 플레이 시간
- 재시작 안내

## 게임 규칙

### 기본 규칙
1. 뱀은 항상 앞으로 이동합니다
2. 먹이를 먹으면 길이가 1칸 늘어납니다
3. 일반 먹이 +10점, Golden Apple +50점
4. 벽 또는 장애물에 부딪히면 게임 오버 (Portal 모드 제외)
5. 자기 몸에 부딪히면 게임 오버

### 게임 오버 조건
- 벽에 충돌 (Portal 모드가 아닐 때)
- 장애물에 충돌 (Hard 난이도)
- 자기 몸통에 충돌

## 팁과 전략

### 초보자
1. **Easy 난이도**로 시작하여 게임에 익숙해지세요
2. **Portal 모드**를 켜면 벽 걱정 없이 플레이 가능
3. 여유 공간을 확보하며 움직이세요

### 일반 플레이어
1. **콤보 유지**: 10초 안에 먹이를 먹어 보너스 점수 획득
2. **Golden Apple 노리기**: 10% 확률이지만 큰 점수
3. **속도 관리**: 먹이를 많이 먹을수록 빨라지니 주의

### 숙련자
1. **Hard 난이도 도전**: 장애물을 피하며 고득점 노리기
2. **공간 활용**: 뱀이 길어질수록 움직일 공간 계획
3. **빠른 반응**: 높은 속도에서도 정확한 컨트롤

## 설정 파일

게임 설정은 `config.json` 파일에 자동 저장됩니다:

```json
{
    "difficulty": "normal",
    "portal_mode": false,
    "sound_enabled": true,
    "music_enabled": true
}
```

게임을 종료하고 다시 시작해도 이전 설정이 유지됩니다.

## 문제 해결

### 게임이 실행되지 않을 때
- Python 3.8 이상 설치 확인
- `pip install -r requirements.txt`로 패키지 재설치
- conda 환경 활성화 확인

### 사운드가 들리지 않을 때
- S 키를 눌러 효과음 켜기
- M 키를 눌러 배경음악 켜기
- 시스템 볼륨 확인

### 게임이 느리게 실행될 때
- 다른 프로그램 종료
- Easy 난이도로 낮추기
- 배경음악 끄기 (M 키)

### 설정이 저장되지 않을 때
- `config.json` 파일 쓰기 권한 확인
- 파일 손상 시 삭제 후 재생성
